# Node JS for demo Pipeline POC.

## Prerequisites

**.npmrc** file should be present in the root directory of the project.
```angular2html
@scope:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=YOUR
```

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```
